# Planes
Control multiple airplanes and try your best to land them without colliding.
### settings.json
```
{    
   "day_born": 3,   
   "month_born": 9, 
   "year_born": 1934,    
   "player_name": "Pippo", 
} 
```
- `day_born, month_born, year_born`: date of birth to greet the user on his birthday
- `player_name`: name of the player
